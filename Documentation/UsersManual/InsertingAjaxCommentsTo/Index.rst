Inserting AJAX comments to page
-------------------------------

.. index::
	single: Setup; Add toctoc_comments to a page in backend

When inserting a new content element, scroll down to the end of "New content element" wizard and
click on the "AJAX Social Network Components":

.. figure:: /Images/image-29.jpg


This will insert a plugin for AJAX Social Network Components as a new content
element. 


Alternatively you can inject the plugin by TS in a TemplaVoila FCE as explained on
the next page.
