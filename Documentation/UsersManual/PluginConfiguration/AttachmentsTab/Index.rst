Attachments tab
^^^^^^^^^^^^^^^

.. figure:: /Images/image-42.jpg 


**Use attachment type webpage preview**  allows or disallows webpage previews when the user is
entering a comment

**Optional webpage preview on top:**  Here you can select an existing webpage preview and put it at
the top of the plug-in

**Index of preview picture** , this is the index number of the image file of the website preview to
display. Check the record in the backend to make your choice first
