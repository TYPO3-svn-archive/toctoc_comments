AJAX Login mode
"""""""""""""""

In this mode the options of the other tabs are not relevant and there are no options to be setup in
the plugin. All options regarding AJAX Login are set up in TypoScript.

Note: TS-options are filed under plugin.tx_toctoc_comments_pi2
