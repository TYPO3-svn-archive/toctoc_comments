Temporary JavaScript files
^^^^^^^^^^^^^^^^^^^^^^^^^^

For some language-dependent strings and some config-values language-dependent JS files are written
and handled during run time. It's also needed for the sharrre-component which is used for sharing.

They are all written in folder res/js/temp/ and can be deleted at any time.
