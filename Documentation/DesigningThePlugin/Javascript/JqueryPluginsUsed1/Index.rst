jQuery plugins used
^^^^^^^^^^^^^^^^^^^

Inputfields with watermark use jQuery Watermark

Textarea auto grow is implemented with (adapted version of) jQuery Elastic

Tooltips and Image presentation is made with (adapted version of) jQuery Tools

Sharing component sharrre is made implemented with jQuery sharrre. (adapted version)

Dialog popups use (adapted version of) jQuery Simple modal (Simpletxtcmoddlg)

Javascript library flowplayer is used for HTML5 video
