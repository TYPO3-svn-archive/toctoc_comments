.. _configuration-setup-options-for-google:

google
^^^^^^

When the options for Google+-login are
setup Google+-login becomes visible in the front end


Note than facebook.imageDir also applies for avatars token from Google+

==============  ==========  ========================================================  ========
**Property:**   Data type:  Description:                                              Default:
--------------  ----------  --------------------------------------------------------  --------
ClientID        string      Google+ ClientID: The ClientID of your Google+
                            application

                            How to make a Google+ App? Login to Google and go
                            https://console.developers.google.com/project?authuser=0
--------------  ----------  --------------------------------------------------------  --------
ClientSecret    wrap        Google+ ClientSecret: The ClientSecret of your Google+
                            application
