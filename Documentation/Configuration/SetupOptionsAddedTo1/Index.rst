toctoc_comments
^^^^^^^^^^^^^^^

All options are setup only and they are
valid per webpage (W).

they are used to configure the comments counter

======================  ==========  ======================================================  ========
**Property:**           Data type:  Description:                                            Default:
======================  ==========  ======================================================  ========
commentsShowCountText   boolean     The number of comments marker is rendered as only
                                    number or as text 
                                    text= 0, number=1
