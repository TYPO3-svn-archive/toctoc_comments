Top activities
""""""""""""""

The activity counter is maintained as sum of, ratings and comments. Ratings can be given more weight
than views, also comments can be weighted differently from views.

The top activities list can be configured: Number of activities required, Number of list items, Days
back to be considered and the target of the top activities: specific records, content, comments or
all of them.

In the screen shot we see the 5 news of a website with most activity.
