Top ratings mode
^^^^^^^^^^^^^^^^

In mode “Top ratings” the plugin displays a list with top ratings with items linked to the
source.

The top ratings can be configured: Number of votes required, Number of list items, Days back to be
considered and the target of the top ratings: specific records, content, comments or all of them.
