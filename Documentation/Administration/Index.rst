Administration
==============

Adding TypoScript template
--------------------------

The extension's TypoScript template must be added to site's TypoScript template.

To add this template, go to **Template**  for the extension, click on the
**Info/Modify**  there and click **Click here to edit whole template record**  link.

In the opened form select Includes, in **Include static (from extensions)**  add
**AJAX Social Network Components (toctoc_comments)**  to selected items.
